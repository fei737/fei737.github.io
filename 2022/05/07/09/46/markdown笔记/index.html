# 使用博客首先要学会熟练的掌握 markdown 编写。

完全内容请参考 (Markdown 官方教程)

# 常用 markdown 的语法。

超链接

直接选中超链接文本快捷键 ctrl+K ，同样超链接文本也能应用粗斜格式。

链接 title 是当鼠标悬停在链接上时会出现的文字，这个 title 是可选的，它放在圆括号中链接地址后面，跟链接地址之间以空格分隔。

使用尖括号可以很方便地把 URL 或者 email 地址变成可点击的链接。

示例：

点击跳转百度

换行

可以直接在 markdown 中回车

但最好可以在一行最后键入两个空格

示例：

我是一个大帅锅

于晏，彦祖你们怎么看？

加粗斜体

可以用快捷键 Ctrl+B 直接加粗选中文本

ctrl+I 可以直接令选中文本变成斜体

同时粗体加斜体，在文本两侧加三个 * 号，或者两个快捷键

示例：

加粗，，斜体，，加粗变斜

选中

快捷键 ctrl+D 直接选择到前面非特殊字符处，适合选择一句话

ctrl+L 可以选中本行所以文字。

引用

引用文本块可以在行前 / 段前加’>' 号即可

块引用还可以包含多个段落，为段落之间的空白行添加一个 > 或者打一个空格。

示例：

爱你们不会变的呦！

这是另一段，不错吧。

图片

要添加图像，请使用感叹号 ( ! ), 然后在方括号增加替代文本，图片链接放在圆括号里，括号里的链接后可以增加一个可选的图片标题文本。

表情

可以键入表情代码，具体参考官方文档 Complete list of github markdown emoji markup

代码块

在一行开头输入三个（`）则可以变成代码编辑框。

行内代码块 CTRL + shift + ~
